## 📌 Sobre o Projeto
O **Screenmatch** é uma aplicação desenvolvida em Java com o objetivo de mapear e aplicar na prática os pilares da Programação Orientada a Objetos (POO). O sistema simula um catálogo de filmes e séries, permitindo a estruturação de dados de entretenimento através de uma arquitetura limpa e escalável.

## 💻 Tecnologias e Conceitos Utilizados
Este projeto explora fundamentos essenciais do ecossistema Java e da engenharia de software:

* **Java** (Linguagem principal)
* **Estruturação de Pacotes** (Organização modular do código)
* **Classes e Variáveis** (Modelagem de domínio)
* **Métodos** (Comportamentos e regras de negócio)
* **Herança** (Reaproveitamento de código entre entidades comuns)
* **Polimorfismo** (Flexibilidade nas chamadas de métodos)
* **Interfaces** (Contratos e padronização de comportamentos)
